inputs, so the whole corpus is redistributable inside this repository.
All sources were retrieved on **2026-07-09**. The committed corpus bytes are
the benchmark's ground truth (`MANIFEST.sha256`); `generate/build_corpus.sh`
documents exactly how each file was produced from the raw sources below.

## Design intent

The mix deliberately mirrors a typical home-user folder: highly compressible
plain text, office documents that are already ZIP containers internally
(docx/xlsx/pptx/epub/odt), already-compressed photos (JPEG/PNG), *un*compressed
raster images (TIFF/BMP — the only image data where compressors can
meaningfully differentiate), and already-compressed H.264 video. Category
subsets let the benchmark report where format choice matters and where it
does not.
